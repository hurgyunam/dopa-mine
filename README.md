# dopa_mine

풀스택 관점에서 제품을 빠르게 만들고 검증하는 개발자입니다.  
TypeScript, React, Flutter, WebView 브릿지 통신까지 한 흐름으로 연결할 수 있고, Cursor AI 기반 바이브 코딩으로 아이디어를 실제 동작하는 기능까지 빠르게 전환합니다.

- Cursor AI를 실무 워크플로우에 녹여 생산성과 실험 속도를 높이는 개발 방식 선호
